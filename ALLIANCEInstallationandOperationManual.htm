1         Overview

The 5W MROU is SOLiDs mid-power, multi-carrier remote optic unit available with ALLIANCE DAS REL6. The unit delivers +37dBm of output power for the following bands:

       700 LTE, 700 PS

       800PS, 800 Sprint, 850 Cellular

       900iDEN (Sprint)/Paging,

       1900 PCS,

       2100 AWS 1+3, and

       2.5GHZ TDD-LTE

 

Support for 2300WCS will be added by Q4 2016. The 5W MROU is equipped with six MRDU bays and can support up to seven different bands simultaneously.

The 5W MROU can also support VHF / UHF bands with the addition of the AOR add-on unit.

NOTE: The 5W MROU will only operate on ALLIANCE REL6 systems. Legacy ALLIANCE Systems can be upgraded to REL6. Contact SOLiD Support for more details.

 

Mix and Match Remote Units in a Single System

The 5W MROU can be mixed with the other ALLIANCE REL6 remote units the low-power 1W LROU and 2W L2ROU or high-power 20W (HROU) in a single DAS. ALLIANCE DAS REL6 Topology

Coupled Test Port

The MROU provides test ports enabling the technician to measure signal strength of downlink (Tx) and uplink (Rx) power without removing the antenna. Test ports on the 5W MROU measure - 30dB below actual downlink/uplink levels.

Auto Shutdown

An Auto Shutdown function shuts down the unit when the composite downlink output power exceeds user-defined values for more than five seconds. This maximum power level can be adjusted using the DMS-1200.

AOR Add-On Unit for VHF/UHF Support

The AOR add-on unit can be connected to the 5W MROU to support VHF / UHF bands.

Power Efficiency

Amplifiers used in the 5W MROU are more power efficient than the current 1W LROU versions, which means the one 5W unit can provide the same RF coverage as five 1W units while using much less power.

For example, a fully-loaded 1W LROU consumes 265W, while the fully-loaded 5W unit consumes 390W. Therefore, it would take five 1W units consuming 1325W (5 x 265W) or about 3.5 times more power to achieve the same RF coverage as one 5W unit, consuming 390W.

Link Budget

Output power at the antenna port is +37dBm for all bands except VHF/UHF, where maximum output power is +24dBm. The System Gain for all bands (except VHF/UHF) in the 5W MROU is 57dB on the downlink and 50dB on the uplink. For VHF/UHF bands, downlink gain is 39dB and uplink gain is 50dB.

The optical link between Donor Optic Units in the ODU or OEU and the remote unit is 10dBo for the 1-port DOU and 5dBo for the 4-port DOU.

 

 

 

DMS-1200 for Monitoring and Control

Using the DMS-1200 REL6, the SOLiD DAS management system, the technician can monitor and control the operation of the MROU as follows:

       Monitor optic power levels and activate optic compensation.

       Monitor temperature levels and other alarm conditions.

       Backup and restore operational settings.

 

MROU Setup Using the DMS-1200

 

Adding the 5W Remote to Legacy ALLIANCE Systems

Note that the 5W MROU will only operate on ALLIANCE REL6 systems. Customers who want to add the new 5W remote to legacy ALLIANCE systems have two options:

1. Upgrade the existing system this entails replacing the MCPU, replacing the OEU (if one exists), upgrading the firmware on all MDBUs, DOUs, RDUs and RCPUs and replacing the DMS. System downtime should be expected with this scenario.

2. Run a parallel system to support a 5W remote, the customer can install and manage a parallel system. This would require a second DMS, BIU, ODU, etc. This scenario will result in little or no system downtime.

3         Design Considerations

Designing with the 5W remote versus 1W can provide the same or better coverage and reduce costs for the customer.

Example: University of Connecticut Ambulatory Care Center

       8 Floors

       303,281 Total Square Feet

       Designed for -80 dBm (pilot) at 95%

       PCS band only

 

1W Design:

       4 Remote Units

       RSSI -80dBm or better achieved in 95.86% of total square footage

       Total UL Noise = 36dB

       System Cost = $155,642

5W Design:

       1 Remote Unit

       RSSI -80dBm or better achieved in 96.26% of total square footage

       Total UL Noise = 34dB

       System Cost = $139,400 (~12% lower than 1W)

4         MROU Configuration Options

The figure shows the components in a fully loaded MROU. Up to six MRDUs can be provisioned in the 5W MROU cabinet.

The MRDUs amplify and adjust both the uplink and downlink RF signals in a power amplifier unit (MRDU). Prior to the downlink signal reaching the MRDU modules, it is first converted from an optic signal into an RF signal and divided into a specific signal type by the R-Optic module. For uplink signals, the unit filters signals received from the combiner in a cavity duplexer, amplifies them and sends them to the R-Optic module. The MRDU automatically adjusts output and input power levels according to user-defined limits.

One combiner (Combiner 1 in the graphic) ships with the MROU. Depending on the number and type of MRDUs used, a second combiner (Combiner 2) can be added.

The unit comes equipped with Band Pass Filtering (BPF) for both the downlink and uplink paths. A separate add-on unit (AOR) is used exclusively for VHF/UHF.

5         5W MROU Specifications

The 5W MROU cabinet satisfies NEMA 4 waterproof, dirt-intrusion, and corrosion-resistance standards. It also satisfies IP66 waterproof and dustproof ratings for electronic equipment enclosures mounted either horizontally or vertically. The enclosure comes equipped with a tamper-proof double lock for installation in unsecured areas.

The unit can operate on AC or DC power input. The cabinet is cooled by a convection cooling design. An optional fan unit is available.
